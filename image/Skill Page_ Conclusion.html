Les vnements DOM

Ajouter des vnements

Les vnements sont des actions qui se produisent la suite d'une action de l'utilisateur ou d'un changement d'tat des lments d'un arbre DOM. Par exemple : click, load, focusin, et focusout. Ce sont tous des vnements qui nous permettent de ragir avec JavaScript.
Nous pouvons assigner des fonctions JS pour couter ces vnements dans les lments et ragir ou rpondre lorsque l'vnement se produit.

Voici une liste d'vnements. Il existe trois faons d'attribuer une fonction un vnement donn.
Si foo() est une fonction personnalise, vous pouvez l'enregistrer en tant qu'couteur d'vnement de clic (elle sera appele lorsque l'lment bouton est cliqu) de trois manires :

// first way
<button onclick=foo>Alert</button>
 // the second way
<div id="main">
 <button>Alert</button>
</div>
<script>
 // Getting the button
 var btn = document.querySelector('button');
 btn.onclick=foo;
</script>
// the third way
<div id="main">
 <button >Alert</button>
</div>
<script>
   // Getting the button
   var btn = document.querySelector('button');
   btn.addEventListener('click', foo);
</script>

Supprimer des vnements

La mthode removeEventListener() dtache un couteur d'vnement (qui a t ajout prcdemment avec la mthode addEventListener()) de l'vnement qu'il coute .

<div id="main">
 <button onclick="foo">Alert</button>
</div>
<script>
     // Getting the button
 let btn = document.querySelector('button');
     // Remove the event listener  
 btn.removeEventListener('click',foo);
</script>

Types d'vnements

Le navigateur dclenche de nombreux vnements. Une liste complte est disponible dans [MDN] (https://developer.mozilla.org/en-US/docs/Web/Events), mais voici quelques-uns des types d'vnements et des noms d'vnements les plus courants :

  • vnements de souris ( MouseEvent ) : mousedown, mouseup, click, dblclick, mousemove, mouseover, mousewheel, mouseout, et contextmenu.
    vnements clavier ( KeyboardEvent ) : keydown, keypress, et keyup
    vnements de formulaire : focus, blur, change, etsubmit
  • vnements de fentre : dfilement, redimensionnement, changement de hash, chargementet dchargement.

Exemple

<!DOCTYPE html>
<html lang="">

<head>
	<meta charset="">
	<meta name="viewport" content="width=, initial-scale=">
	<title></title>
</head>

<body>
	<div class="d-flex h-100">
		<p>try pressing some keys:</p>
		<p id="log"></p>
	</div>
	<script>
		document.addEventListener('keydown', logKey);
    	function logKey(event) {
      	log.textContent += " " + e.code;
    	}
	</script>
</body>

</html>
