Les évènements DOM

Les évènements DOM

Ajouter des événements

Les événements sont des actions qui se produisent à la suite d'une action de l'utilisateur ou d'un changement d'état des éléments d'un arbre DOM. Par exemple : click, load, focusin, et focusout. Ce sont tous des événements qui nous permettent de réagir avec JavaScript.
Nous pouvons assigner des fonctions JS pour écouter ces événements dans les éléments et réagir ou répondre lorsque l'événement se produit.

Voici une liste d'événements. Il existe trois façons d'attribuer une fonction à un événement donné.
Si foo() est une fonction personnalisée, vous pouvez l'enregistrer en tant qu'écouteur d'événement de clic (elle sera appelée lorsque l'élément bouton est cliqué) de trois manières :

// first way
<button onclick=foo>Alert</button>
 // the second way
<div id="main">
 <button>Alert</button>
</div>
<script>
 // Getting the button
 var btn = document.querySelector('button');
 btn.onclick=foo;
</script>
// the third way
<div id="main">
 <button >Alert</button>
</div>
<script>
   // Getting the button
   var btn = document.querySelector('button');
   btn.addEventListener('click', foo);
</script>

Supprimer des événements

La méthode removeEventListener() détache un écouteur d'événement (qui a été ajouté précédemment avec la méthode addEventListener()) de l'événement qu'il écoute .

<div id="main">
 <button onclick="foo">Alert</button>
</div>
<script>
     // Getting the button
 let btn = document.querySelector('button');
     // Remove the event listener  
 btn.removeEventListener('click',foo);
</script>

Types d'événements

Le navigateur déclenche de nombreux événements. Une liste complète est disponible dans [MDN] (https://developer.mozilla.org/en-US/docs/Web/Events), mais voici quelques-uns des types d'événements et des noms d'événements les plus courants :

  • Événements de souris ( MouseEvent ) : mousedown, mouseup, click, dblclick, mousemove, mouseover, mousewheel, mouseout, et contextmenu.
    Événements clavier ( KeyboardEvent ) : keydown, keypress, et keyup
    Événements de formulaire : focus, blur, change, etsubmit
  • Événements de fenêtre : défilement, redimensionnement, changement de hash, chargementet déchargement.

Exemple

<!DOCTYPE html>
<html lang="">

<head>
	<meta charset="">
	<meta name="viewport" content="width=, initial-scale=">
	<title></title>
</head>

<body>
	<div class="d-flex h-100">
		<p>try pressing some keys:</p>
		<p id="log"></p>
	</div>
	<script>
		document.addEventListener('keydown', logKey);
    	function logKey(event) {
      	log.textContent += " " + e.code;
    	}
	</script>
</body>

</html>
